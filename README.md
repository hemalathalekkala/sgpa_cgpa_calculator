# SGPA & CGPA Calculator

## Description

This is a simple Python project that calculates **SGPA** for each semester and the **final CGPA** based on the grades and credits entered by the user.

## Features

* Calculate SGPA for each semester
* Calculate final CGPA
* Accept grades and credits as input
* Display student performance

## Technologies Used

* Python

## How to Run

1. Download the project.
2. Open the file in Python.
3. Run the program.
4. Enter the required grades and credits.

## Grade Scale

* O = 10
* A+ = 9
* A = 8
* B+ = 7
* B = 6
* C = 5
* D = 4
